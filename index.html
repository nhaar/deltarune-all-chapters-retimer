Info

A tool to calculate Deltarune All Chapter times

To use it, pick your method of getting timestamps from the video (see "Retimer"), and add the proper timestamps in the "Times" field below

Retimer

Please use one of the verified retiming tools, as they are confirmed to be as accurate as possible:

  • Vidtimer - Make sure to set the proper video framerate, and copy the "ff:ss" time
  • Somewes - Make sure to set the proper video framerate, copy either the "Frame Time" or the "Current Frame" field

Supported Formats

These are the allowed formats you can input in the fields below:

  • A raw frame count, which will be detected if you only give nothing but a number
  • Time in the format "ss:ff" (seconds: frames)
  • Time in the format "hh:mm:ss.decimal" (hours, minutes, seconds, decimals), with hours and minutes being optional
